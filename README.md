# todo-list
 A todo list web application for users to enter all the things they have to do on a specific day

Visual Studio Code
MAMP
PHP
Apache Servers
Javascript

User enters an item which is a thing to do and then it displays on the screen
If a user is done with an item they can just click on it and it will strike out that item and can unstrike too
If a user has a completed a task they can delete it from the list
